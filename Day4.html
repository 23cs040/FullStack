ABOUT CSS

Inline CSS

Inline CSS is a method of applying unique CSS styles.

Internal CSS

Internal CSS, also known as embedded CSS, is a method of defining css rules directly within a single HTML document using the style tag.

EXternal CSS

External CSS is a method of styling multiple HTML pages by using a single, separate CSS file.
